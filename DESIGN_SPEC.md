# AI 교통·내비게이션 앱 프로토타입 설계 문서

## 프로젝트 개요
차량번호 입력 기반 실시간 추적 + AI 내비게이션을 통합한 모바일 앱 프로토타입

---

## 화면별 상세 명세

### 1. 시작 화면 (Splash Screen)
**목적**: 앱 진입 및 차량 선택

**UI 요소**:
- 앱 로고 (상단 중앙)
- 앱 이름: "AI Navigator"
- 차량번호 입력 필드
- "차량번호 입력" 버튼 (Primary)
- "내 차량 선택" 버튼 (Secondary)

**인터랙션**:
- 차량번호 입력 시 자동 포맷팅 (123가4567)
- 버튼 클릭 → 지도 화면으로 슬라이드 전환

---

### 2. 지도 화면 (Map View)
**목적**: 실시간 위치 추적 및 차량 모니터링

**UI 요소**:
- 지도 영역 (전체 화면 배경)
- 상단 바:
  - 차량번호 표시
  - GPS 신호 아이콘
  - 현재 속도 표시 (km/h)
- 중앙: 현재 위치 마커
- 하단 카드:
  - "따라가기 모드" 토글 스위치
  - "내비게이션 시작" 버튼
- 플로팅 버튼: 현재 위치로 이동

**인터랙션**:
- 지도 드래그/줌 가능
- 따라가기 토글 ON → 지도 자동 중심 맞춤
- "내비게이션 시작" → 내비게이션 화면으로 전환

---

### 3. 내비게이션 화면 (Navigation Screen)
**목적**: 목적지 설정 및 AI 경로 계산

**UI 요소**:
- 상단:
  - 출발지 (현재 위치, 자동 입력)
  - 목적지 입력 필드 (검색 가능)
- 중앙: 경로 미리보기 지도
- 하단 카드:
  - "AI 경로 계산" 버튼
  - 예상 거리
  - 예상 시간 (ETA)
  - 경로 옵션 (최단거리/최단시간/AI추천)

**인터랙션**:
- 목적지 검색 → 자동완성 제안
- AI 경로 계산 → 로딩 애니메이션 → 경로 표시
- "안내 시작" 버튼 → 결과 화면으로 전환

---

### 4. 결과 화면 (Summary Screen)
**목적**: 주행 완료 후 통계 및 피드백

**UI 요소**:
- 상단: 성공 아이콘 ✓
- "목적지 도착!" 메시지
- 통계 카드:
  - 총 주행 거리
  - 총 주행 시간
  - 평균 속도
  - AI 예측 정확도
- "다시 시작" 버튼
- "홈으로" 버튼

**인터랙션**:
- "다시 시작" → 지도 화면으로
- "홈으로" → 시작 화면으로

## 애니메이션 명세

### 화면 전환
- **타입**: 좌우 슬라이드
- **지속시간**: 300ms
- **Easing**: cubic-bezier(0.4, 0.0, 0.2, 1)

### 버튼 인터랙션
- **Hover**: Scale 1.02, 그림자 증가
- **Active**: Scale 0.98
- **지속시간**: 150ms

### 로딩 애니메이션
- **타입**: 원형 프로그레스
- **색상**: Primary Blue
- **크기**: 40px
