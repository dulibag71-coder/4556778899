# AI Navigator - 스마트 내비게이션 프로토타입

차량번호 입력 기반 실시간 추적 + AI 내비게이션 통합 앱 프로토타입

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![Status](https://img.shields.io/badge/status-prototype-orange)

## 프로젝트 개요

AI Navigator는 차량 추적과 AI 기반 경로 계산을 하나의 플로우로 통합한 모바일 웹 애플리케이션 프로토타입입니다.

### 주요 기능

- **차량 추적**: 차량번호 입력으로 실시간 GPS 위치 추적
- **지도 연동**: OpenStreetMap 기반 인터랙티브 지도
- **AI 경로 계산**: 최단거리, 최단시간, AI 추천 경로 옵션
- **따라가기 모드**: 자동으로 차량 위치 중심 맞춤
- **주행 통계**: 거리, 시간, 속도, AI 정확도 등 분석

## 화면 구성

### 1. 시작 화면 (Splash Screen)
- 앱 로고 및 이름
- 차량번호 입력 필드
- "차량번호 입력" / "내 차량 선택" 버튼

### 2. 지도 화면 (Map View)
- 실시간 지도 (OpenStreetMap)
- 현재 위치 마커 및 GPS 상태
- 속도 표시 (km/h)
- 따라가기 모드 토글
- "내비게이션 시작" 버튼

### 3. 내비게이션 화면 (Navigation)
- 출발지/목적지 입력
- 경로 미리보기 지도
- 경로 옵션 선택 (최단거리/최단시간/AI추천)
- AI 경로 계산 기능
- 예상 거리 및 ETA 표시

### 4. 결과 화면 (Summary)
- 도착 완료 메시지
- 주행 통계 (거리, 시간, 속도, AI 정확도)
- "다시 시작" / "홈으로" 버튼

## 사용 방법

### 로컬에서 실행

1. 프로젝트 파일을 다운로드합니다:
```bash
git clone <repository-url>
cd 4556778899
```

2. 웹 서버를 시작합니다 (Python 예시):
```bash
# Python 3
python -m http.server 8000

# 또는 Python 2
python -m SimpleHTTPServer 8000
```

3. 브라우저에서 접속합니다:
```
http://localhost:8000
```

### VS Code Live Server 사용

1. VS Code에서 `index.html` 파일을 엽니다
2. 우클릭 → "Open with Live Server" 선택
3. 자동으로 브라우저가 열립니다

### 직접 파일 열기

최신 브라우저에서 `index.html` 파일을 직접 열어도 실행됩니다.

## 사용자 플로우

```
시작 → 차량번호 입력 → 지도 보기 → 내비게이션 설정 → 경로 안내 → 결과 확인
```

### 단계별 가이드

1. **차량번호 입력**
   - 차량번호를 직접 입력하거나 "내 차량 선택" 클릭
   - 예: `12가3456`

2. **지도 화면**
   - GPS 신호와 현재 속도 확인
   - 따라가기 모드 ON/OFF 전환
   - 지도 드래그/줌 가능
   - "내비게이션 시작" 클릭

3. **경로 설정**
   - 목적지 입력 (자동완성 지원)
   - 경로 옵션 선택
   - "AI 경로 계산" 클릭
   - 예상 거리/시간 확인 후 "안내 시작"

4. **결과 확인**
   - 주행 통계 확인
   - "다시 시작" 또는 "홈으로" 선택

이 프로토타입을 Figma 디자인으로 변환하려면:

1. **FigJam에서 플로우 작성**
   - `DESIGN_SPEC.md`의 플로우 다이어그램 참고
   - 4개 화면 연결 (시작 → 지도 → 내비게이션 → 결과)

2. **Figma에서 프레임 생성**

   - 각 화면을 별도 프레임으로 생성

   - Color Styles: 명세서의 색상 팔레트 등록

   - Component: 버튼, 카드, 입력 필드 컴포넌트화

4. **프로토타입 연결**
   - Smart Animate: 화면 전환
   - Transition: 300ms, Ease Out
