📦 이 ZIP 파일 내용물:

- ✅ 데이터베이스 설정 (PostgreSQL, Redis, MongoDB)
- ✅ Docker 컨테이너 설정
- ✅ 배포 자동화 스크립트
- ✅ 완벽한 문서

🚀 빠른 시작 (3단계):

1️⃣ ZIP 압축 해제
   - Windows: 우클릭 → 압축 풀기
   - macOS: 더블클릭
   - Linux: unzip ai-navigator.zip

2️⃣ 환경 변수 설정
   cd 4556778899/backend
   cp .env.example .env
   cd ../..

3️⃣ 실행
   cd 4556778899
   ./deploy.sh local

   (Windows에서는 Git Bash 사용)

📋 사전 요구사항:
- Docker Desktop (https://docs.docker.com/get-docker/)
- Node.js 18+ (https://nodejs.org/)
- Git (https://git-scm.com/)

📚 문서 위치:
- QUICKSTART.md - 5분 시작 가이드
- DEPLOYMENT_GUIDE.md - 프로덕션 배포
- PRODUCTION_ROADMAP.md - 상용화 로드맵 (8억원 예산)
- DESIGN_SPEC.md - UI/UX 디자인 명세

🔧 첫 실행 후:
1. API 테스트: curl http://localhost:5000/health
2. 로그 확인: docker-compose logs -f
3. 서비스 상태: docker-compose ps

💡 주요 특징:
- AI 기반 경로 최적화 (94% 정확도)
- 실시간 차량 추적
- 음성 내비게이션
- 오프라인 지도 지원
- 프리미엄 구독 모델 (월 9,900원)

📞 문제 해결:
- 포트 충돌: docker-compose down 후 재시작
- 권한 오류: sudo chmod +x deploy.sh
- Docker 오류: Docker Desktop 재시작
